About Me

I'm a cloud security professional with active CompTIA Security+, CySA+, Network+ certifications and the Google Cybersecurity Professional Certificate. My background combines hands-on AWS security experience with foundational knowledge in security operations and DoD compliance frameworks.

I'm currently building cloud security projects to apply what I've learned:

  • AWS IAM hardening and policy design
  • Terraform modules with secure defaults
  • RMF control mapping for AWS services
  • CloudTrail and GuardDuty monitoring

My professional focus is on cloud security engineering with expertise in AWS services, infrastructure as code, and compliance frameworks (RMF, NIST 800-53). I'm working toward these through a structured learning plan that builds on my current certifications and hands-on projects.

I hold an expired AWS Certified Cloud Practitioner certification and am currently studying for the AWS Solutions Architect – Associate (SAA-C03) certification as my next step.

The name Imago Dei reflects my belief that humans are created with purpose, creativity, and responsibility. I approach technology with that same mindset: to build with integrity, to solve problems thoughtfully, and to contribute to a safer, more resilient digital landscape.
