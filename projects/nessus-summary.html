Vulnerability Management · Nessus Scan Summary

Executive Summary (One Page)

Nessus scan against Metasploitable target identified 47 vulnerabilities. Top 3 findings by CVSS score: Apache Tomcat default credentials (9.8), Unpatched SMB vulnerability (9.8), and Weak SSH configuration (7.5). Remediation plans developed for all three, mapped to NIST controls. Critical findings require immediate remediation; high findings within 30 days.

Scan Environment

  • Target: Metasploitable 2 (vulnerable Linux VM)
  • Scanner: Nessus Essentials (Tenable)
  • Scan Type: Unauthenticated network scan
  • Scan Date: April 2026
  • Environment: AWS EC2 (isolated VPC)

Top 3 Findings by CVSS

Finding 1: Apache Tomcat Default Credentials

CVSS Score: 9.8 (Critical)

Description: Default Tomcat administrator credentials (admin/admin) are enabled, allowing unauthenticated remote access to the management interface.

Remediation: Remove default accounts immediately. Implement strong password policy for all administrative accounts. Restrict Tomcat manager access to trusted IP ranges using security groups.

NIST Mapping: CM-6 (Configuration Settings), IA-2 (Identification & Authentication)

Finding 2: Unpatched SMB Vulnerability (MS17-010 / EternalBlue)

CVSS Score: 9.8 (Critical)

Description: The SMBv1 protocol is enabled and missing critical security patches, making the host vulnerable to remote code execution exploits.

Remediation: Apply Microsoft security patch MS17-010. Disable SMBv1 if not required. Segment vulnerable systems from production networks.

NIST Mapping: CM-2 (Baseline Configuration), SI-3 (Malicious Code Protection)

Finding 3: Weak SSH Configuration

CVSS Score: 7.5 (High)

Description: SSH server allows password authentication without MFA and permits root login, increasing risk of unauthorized access.

Remediation: Disable root SSH login. Enforce key-based authentication. Implement MFA for all SSH access. Configure SSH to use non-standard port and implement fail2ban.

NIST Mapping: AC-3 (Access Enforcement), IA-2 (Identification & Authentication)
