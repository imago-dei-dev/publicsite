Why This Project

My AWS IAM Foundation project showed me that manual configuration and auditing doesn't scale. To work effectively in cloud security and RMF engineering, I need automation skills. This project documents my journey learning Python and boto3 from scratch.

Learning Plan

Phase 1: Python Fundamentals (March - April 2026)

  • Weeks 1-2: Syntax, variables, data types, conditionals
  • Weeks 3-4: Loops, functions, basic file I/O
  • Weeks 5-6: Error handling, modules, simple scripts

Phase 2: boto3 Introduction (April - May 2026)

  • AWS SDK for Python setup and configuration
  • Making basic API calls (list buckets, describe instances)
  • Understanding pagination and error handling in boto3

Phase 3: Evidence Collection Tools (June 2026 onward)

  • Building modular scripts for compliance evidence
  • JSON output formatting for audit readiness
  • Integration with IAM and other AWS services
